# components playground

这个 playground 只用于 `@moluoxixi/components` 包的组件示例和交互验证。

## 边界

- 这里展示 `packages/components` 暴露的组件，例如 `DateRangePicker`、`EnterNextContainer`、`PopoverTableSelect`。
- 不在这里承载 ConfigForm 对第三方 UI 框架的接入示例。
- Element Plus、Ant Design Vue、shadcn-vue 等 ConfigForm 接入示例分别放在独立 playground 中。
